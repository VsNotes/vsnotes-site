🌟 Artificial Intelligence (AI)
(Goal: Make machines act smart like humans)
|
|-- Starting point: "Give instructions to machines"
|
├── 🤔 Non‑Learning AI (Old‑School Brain)
│   (Machines follow rules, but never improve)
│   |
│   ├── Rule‑Based Systems
│   │   → “IF this happens, DO that”
│   │   → Human writes every possible rule
│   |
│   ├── Logic Engines
│   │   → Formal reasoning & constraints
│   │   → Logic works, but learning does not happen
│   |
│   └── Automation Scripts
│       → Fast, dumb, predictable helpers
│       → Good for repetition, bad for intelligence
│
│   ❌ Problem:
│   • Same mistakes repeat
│   • No learning from experience
│   • No adaptation to new situations
│
│   ✅ This limitation CREATED the need for learning systems
│
├── 📈 Machine Learning (ML)
│   (Machines learn from experience instead of fixed rules)
│   |
│   |-- Key shift:
│   |   Rules → Data
│   |   Instructions → Examples
│
│   ├── 🧩 Classical Machine Learning
│   │   (Humans still guide what to learn)
│   │   |
│   │   ├── Human‑made Features
│   │   │   → “Tell the machine what to look for”
│   │   │   → Human intelligence still dominant
│   │   |
│   │   ├── Core Tasks
│   │   │   → Regression (predict numbers)
│   │   │   → Classification (predict labels)
│   │   |
│   │   └── Common Models
│   │       → Decision Trees
│   │       → SVM
│   │       → k‑NN
│   │
│   │   ❌ Limitation:
│   │   • Humans still design features
│   │   • Complex data (images, speech) breaks system
│   │
│   │   ✅ This limitation CREATED the need for deeper learning
│
│   └── 🧠 Deep Learning (DL)
│       (Machines learn features by themselves)
│       |
│       |-- Big shift:
│       |   Human features → Machine‑learned features
│
│       ├── 🔗 Neural Networks (The Brain Core)
│       │   |
│       │   ├── Input Layer
│       │   │   → Receives raw data
│       │   |
│       │   ├── Hidden Layers
│       │   │   → Learn patterns step‑by‑step
│       │   |
│       │   └── Output Layer
│       │       → Final prediction
│       |
│       ├── 👁️ CNN – Convolutional Neural Networks
│       │   |
│       │   ├── Sees edges → shapes → objects
│       │   ├── Convolution Layers
│       │   ├── Pooling Layers
│       │   └── Fully Connected Layers
│       │
│       │   ✅ Perfect for images & video
│       |
│       ├── 🔁 RNN / LSTM
│       │   |
│       │   └── Handles sequences
│       │       → time, speech, text order
│       |
│       └── ⚡ Transformers
│           |
│           ├── Attention
│           │   → What matters most?
│           |
│           ├── Self‑Attention
│           │   → Words talk to words
│           |
│           └── Multi‑Head Attention
│               → Many views at same time
│
├── 🗣️ Natural Language Processing (NLP)
│   (Teaching machines human language)
│   |
│   |-- Built on top of:
│   |   Deep Learning + Transformers
│
│   ├── Text Understanding
│   ├── Speech Understanding
│   └── Meaning & Intent Detection
│
├── 🧠 Large Language Models (LLMs)
│   (Language intelligence at massive scale)
│
├── 👀 Computer Vision (CV)
│
└── 🧩 RAG (Retrieval‑Augmented Generation)
